# Landing Page – Fitness Marketing Theme

## Overview
This project is a landing page built as part of **The Odin Project** curriculum.  
The design is inspired by **gym and fitness social media marketing agencies**, focusing on strong visual hierarchy and conversion-oriented layout.

## Technologies Used
- HTML5  
- CSS3 (Flexbox)

## Features
- Flexbox-based layout
- Sectioned landing page structure
- Emphasis on typography and spacing
- Marketing-style call-to-action sections
